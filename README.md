# ☕ Coffee Shop Sales Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Coffee Shop Sales Dashboard created in **Microsoft Excel**. The dashboard helps analyze sales performance, customer trends, and product insights using Pivot Tables, Pivot Charts, Slicers, and Excel formulas.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Track monthly sales trends.
* Compare sales across different store locations.
* Identify top-selling product categories.
* Create an interactive dashboard for business analysis.

---

## 🛠 Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Formulas
* Conditional Formatting
* Data Cleaning
* Dashboard Design

---

## 📊 Dashboard Includes

* Total Sales
* Total Orders
* Total Quantity Sold
* Monthly Sales Analysis
* Sales by Store Location
* Sales by Product Category
* Sales by Product Type
* Interactive Slicers for easy filtering
